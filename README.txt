"# DOS_DDOS_detection" 
This Python script leverages the Scapy library to monitor network traffic in real-time, specifically looking for potential DoS (Denial of Service) attacks by analyzing packet rates from various source IP addresses. The script defines a threshold (set to 40 packets per second) to identify suspicious activity and includes an exclusion list to prevent blocking the machine's own IP. When an IP exceeds the defined threshold, the script automatically blocks it using the Windows Firewall command netsh advfirewall, while also logging the blocked IP and its packet rate in a designated log file (blocked_ips.log). The script runs continuously, processing incoming packets and utilizing a callback function to track packet counts within one-second intervals. Additionally, it checks for administrative privileges at startup, ensuring proper execution of firewall commands.

Ce script Python utilise la bibliothèque Scapy pour surveiller le trafic réseau en temps réel, en recherchant spécifiquement des attaques potentielles par déni de service (DoS) en analysant les taux de paquets provenant de différentes adresses IP sources. Le script définit un seuil (fixé à 40 paquets par seconde) pour identifier une activité suspecte et inclut une liste d'exclusion afin d'éviter de bloquer l'adresse IP de la machine elle-même. Lorsqu'une IP dépasse le seuil défini, le script la bloque automatiquement en utilisant la commande de pare-feu Windows netsh advfirewall, tout en enregistrant l'IP bloquée et son taux de paquets dans un fichier journal désigné (blocked_ips.log). Le script fonctionne en continu, traitant les paquets entrants et utilisant une fonction de rappel pour suivre les comptes de paquets dans des intervalles d'une seconde. De plus, il vérifie les privilèges administratifs au démarrage, garantissant l'exécution correcte des commandes de pare-feu.
